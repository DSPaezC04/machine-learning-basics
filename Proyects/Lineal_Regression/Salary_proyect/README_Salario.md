# 💼 Predicción de Salario con Regresión Lineal

## 📌 Descripción

Este proyecto implementa un modelo de **regresión lineal** para predecir el salario de una persona en función de sus años de experiencia.

El objetivo es entender cómo funciona un modelo básico de Machine Learning y cómo puede aprender relaciones entre datos.

---

## 🧠 Conceptos aplicados

* Regresión lineal
* División de datos (train/test)
* Entrenamiento de modelo
* Predicción
* Visualización de datos

---

## 📊 Datos utilizados

Se utilizaron datos simples creados manualmente:

* Años de experiencia (variable independiente)
* Salario (variable dependiente)

Esto permite enfocarse en el funcionamiento del modelo sin depender de datasets complejos.

---

## ⚙️ Tecnologías

* Python
* NumPy
* Matplotlib
* Scikit-learn

---

## 🔍 Flujo del proyecto

1. Creación de datos
2. División en entrenamiento y prueba
3. Entrenamiento del modelo de regresión lineal
4. Predicción de nuevos valores
5. Visualización de resultados

---

## 📈 Ejemplo de predicción

El modelo puede estimar el salario para nuevos valores.
Por ejemplo:

* Entrada: 7 años de experiencia
* Salida: salario estimado por el modelo
