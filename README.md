# Machine Learning Basics

## Overview

This repository documents my learning process in Machine Learning, focusing on building a solid understanding of fundamental concepts through hands-on implementation in Python.

The projects included here are designed to explore how machine learning models work from the ground up, with an emphasis on data preparation, model training, evaluation, and interpretation of results.

---

## Objectives

* Develop a strong foundation in Machine Learning concepts
* Understand how models learn from data
* Apply algorithms to real and synthetic datasets
* Build clean, readable, and well-documented code
* Create a structured portfolio of incremental learning projects


---

## Projects

### 1. CO2 Emissions Prediction

This project applies linear regression to a real-world dataset containing vehicle information.

**Key aspects:**

* Data exploration and visualization
* Feature selection
* Model training using linear regression
* Performance evaluation using standard metrics

The goal is to understand how different vehicle characteristics influence CO2 emissions.

---

### 2. Salary Prediction

This project uses a simple synthetic dataset to model the relationship between years of experience and salary.

**Key aspects:**

* Manual data creation
* Model training and prediction
* Visualization of regression results

This project focuses on understanding the core mechanics of linear regression without external data complexity.

---

## Methodology

All projects in this repository follow a consistent workflow:

1. Data collection or creation
2. Data exploration and visualization
3. Feature selection
4. Model training
5. Model evaluation
6. Interpretation of results

This structured approach ensures clarity and reproducibility.
